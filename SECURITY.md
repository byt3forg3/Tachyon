# Security Policy

## Supported Versions

Tachyon Hash Family is explicitly designed as an **experimental** high-throughput hash function. There are currently no formally audited or officially supported versions for production use.

### Tachyon Core

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | ❌ (Experimental)  |
| < 0.1   | ❌                 |

> **Note:** For a detailed breakdown of Tachyon Core's design, threat model, and statistical verification (SMHasher, PractRand, etc.), please read the [Algorithm Security Documentation](algorithms/tachyon/SECURITY.md).

### Tachyon Zero (In Development)

| Version | Supported          |
| ------- | ------------------ |
| N/A     | ❌                 |
